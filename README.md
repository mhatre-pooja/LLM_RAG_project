# Reducing Hallucination using RAG

## Objective
This project aims to reduce hallucination in Large Language Models using Retrieval-Augmented Generation (RAG) with retrieval optimization and reranking techniques.

## Features
- PDF document retrieval
- Semantic chunking
- FAISS vector database
- Embedding generation
- CrossEncoder reranking
- Hallucination evaluation
- Accuracy and latency analysis

## Technologies Used
- LangChain
- FAISS
- Sentence Transformers
- OpenAI GPT
- Python

## Methodology
1. Upload PDF
2. Create chunks
3. Generate embeddings
4. Store vectors in FAISS
5. Retrieve relevant chunks
6. Apply reranking
7. Generate final response

## Results
- Improved retrieval quality
- Reduced hallucination
- Better contextual grounding
- Higher accuracy
